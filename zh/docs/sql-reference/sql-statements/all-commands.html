    用户账户管理​

    管理用户、角色、和权限。

    • ALTER USER
    • CREATE ROLE
    • CREATE USER
    • DROP ROLE
    • DROP USER
    • EXECUTE AS
    • GRANT
    • REVOKE
    • SET DEFAULT ROLE
    • SET PASSWORD
    • SET PROPERTY
    • SET ROLE
    • SHOW AUTHENTICATION
    • SHOW GRANTS
    • SHOW PROPERTY
    • SHOW ROLES
    • SHOW USERS

    集群管理​

    管理集群，包括 FE、BE、Compute Node (CN)、资源组 (Resource Group)、存储卷（Storage Volume）、表、Tablet、副本 (Replica)、文件、索引（Index）、变量（Variable）、插件（Plugin）等。

    FE，BE，CN，Broker，process​

    • ADMIN SET CONFIG
    • ADMIN SHOW CONFIG
    • ALTER SYSTEM
    • CANCEL DECOMMISSION
    • KILL
    • SHOW BACKENDS
    • SHOW BROKER
    • SHOW COMPUTE NODES
    • SHOW FRONTENDS
    • SHOW PROC
    • SHOW PROCESSLIST

    资源组​

    • ALTER RESOURCE GROUP
    • DROP RESOURCE GROUP
    • SHOW RESOURCE GROUP

    存储卷​

    • ALTER STORAGE VOLUME
    • CREATE STORAGE VOLUME
    • DESC STORAGE VOLUME
    • DROP STORAGE VOLUME
    • SET DEFAULT STORAGE VOLUME
    • SHOW STORAGE VOLUMES

    表，tablet，副本​

    • ADMIN CANCEL REPAIR TABLE
    • ADMIN CHECK TABLET
    • ADMIN REPAIR TABLE
    • ADMIN SET REPLICA STATUS
    • ADMIN SHOW REPLICA DISTRIBUTION
    • ADMIN SHOW REPLICA STATUS
    • RECOVER
    • SHOW TABLE STATUS

    文件，索引，变量​

    • CREATE FILE
    • CREATE INDEX
    • DROP FILE
    • DROP INDEX
    • SET (variable)
    • SHOW FILE
    • SHOW FULL COLUMNS
    • SHOW INDEX
    • SHOW VARIABLES

    • SHOW SQLBLACKLIST
    • DELETE SQLBLACKLIST

    插件​

    • INSTALL PLUGIN
    • SHOW PLUGINS
    • UNINSTALL PLUGIN
